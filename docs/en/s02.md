# Getting Started with DevRev

> "Master the big picture in your first 30 minutes. That changes the speed of everything after."

## The Problem

When you first log in to DevRev, the left sidebar is packed with menu items. It is unclear where to start. Documentation is primarily in English, and there are many configuration options. Many people give up during this initial experience.

Users accustomed to Jira or Zendesk often find DevRev's navigation structure confusing because it is organized differently.

## The Solution

What you need to do first comes down to three steps:

1. **Create a workspace**: In DevRev, a "workspace" is the organizational unit. Create one first
2. **Set up your profile**: Configure your account so you are recognized as a team member
3. **Understand basic navigation**: Learn the left sidebar structure so you can access key screens without confusion

Completing these three steps in your first 30 minutes is the key to accelerating your learning.

## How It Works

DevRev's interface is organized around the left sidebar.

Key navigation elements include:

- **Home**: Dashboard and activity feed
- **Inbox**: List of tickets and issues assigned to you
- **Tickets**: Support ticket management (Support App)
- **Issues**: Development issue management (Build App)
- **Parts**: Product structure definition and management
- **Settings**: Workspace-wide configuration

DevRev provides search and keyboard shortcuts. The search bar at the top lets you search across tickets, issues, and parts.

When creating a workspace, you configure the following:

| Setting | Description |
|---------|-------------|
| Workspace name | Organization or project name |
| Timezone | Affects SLA policy calculations |
| Default language | UI and template language |

## What You Will Learn

**Before this session:**
- You created an account but do not know what to do after logging in
- You have not grasped the sidebar menu structure
- You are unaware of search and shortcuts

**After this session:**
- Your workspace is created and profile is set up
- You can explain the role of each major sidebar menu (Home / Inbox / Tickets / Issues / Parts)
- You can quickly access objects using the search bar
